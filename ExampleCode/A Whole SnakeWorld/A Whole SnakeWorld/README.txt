INSTALLATION NOTES
	- For running this game download and install XNA 4.0 in http://create.msdn.com/en-US/

REQUIREMENTS
	- Make sure you have an excellent graphics card to run this game! (XNA HiDef property)

PLAYING THE GAME
	- Click on A_Whole_SnakeWorld.ccgames (located in /bin) to start the game
	- SPACE to jump. Have fun!
